# Sales Training Platform

Interactive sales training platform focused on AWS methodologies: Challenger, MEDDIC, and SPIN.

## Features

- User authentication (signup/login)
- Interactive learning modules
- Gamified progress tracking
- Practice scenarios with feedback
- AWS-specific talk tracks

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open browser to the URL shown (typically http://localhost:5173)

## Structure

- `/src/pages` - Main application pages (Login, Dashboard, Module, Practice)
- `/src/data` - Content for modules and practice scenarios
- `/src/styles` - CSS styling for each component
